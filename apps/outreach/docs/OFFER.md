# Freightdev Offer

## Core promise

Freightdev builds websites, landing pages, forms, and simple operating systems for trucking companies that need to look professional, capture leads, and stop losing work in texts, PDFs, and scattered inboxes.

## Starter package: Landing Page + Lead Form

Best for owner-operators, small carriers, dispatchers, recruiters, repair shops, or trucking-adjacent businesses with weak/no web presence.

Includes:

- Mobile-first one-page site
- Services/equipment/lane copy
- Click-to-call and email actions
- Project/quote/contact form
- Resend email notification setup
- Basic deployment support

## Growth package: Website + Intake System

Includes Starter plus:

- Multiple pages
- Driver application or shipper/broker quote form
- Local/CRM lead tracking
- Follow-up templates
- Simple dashboard/reporting path

## Custom system package

For messy operations that need a real workflow:

- CRM or lead board
- Driver application workflow
- Document upload/intake
- Customer request portal
- Dispatch/customer support tools
- Email/SMS automation where appropriate

## Selling rule

Do not sell generic AI. Sell the trucking outcome: credibility, leads, faster follow-up, cleaner intake, and less manual chaos.
