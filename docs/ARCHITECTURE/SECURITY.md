# Security Architecture

Default trust is zero.

No user, actor, device, external content, memory, or voice is trusted automatically.

## Protected access path

1. Key Keeper checks authority.
2. Secret Safe verifies and controls secret access.
3. Unit Usage controls disclosure.
4. Memory Mark validates untrusted content.
5. Ghost Guard monitors behavior.
6. Legal Logger records evidence.

## Ownership

- Key Keeper owns authorization.
- Secret Safe owns protected secrets.
- Unit Usage owns personal privacy disclosure.
- Memory Mark owns unsafe memory validation.
- Ghost Guard owns integrity monitoring.
- Legal Logger owns evidence.

## Rule

Security must be structural, not only prompted.
