# UI Architecture

Do not build only a chat app.

Build an AI operations console.

CoDriver is the first console for the runtime.

## Mobile-first navigation

- Me.
- Connect.
- AI.
- Loads.
- More.

## CoDriver wake-up state

CoDriver should wake up to operational status, not an empty chat box.

The UI should display:

- Actor status.
- System health.
- Active loads.
- Pending packets.
- Flow activity.
- Action usage.
- Recent reports.
- Timeline views.
- Logbook entries.

## Rule

The UI consumes runtime state.

The UI is not the source of truth.
