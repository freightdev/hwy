# Logbook Architecture

Everything important has a Logbook.

Logbooks preserve history.

Legal Logger preserves truth.

## Logbook law

Logbooks are append-only.

Do not silently rewrite history.

Corrections create new entries.

## Examples

- Load Logbook.
- Flow Logbook.
- Actor Logbook.
- Agent Logbook.
- Worker Logbook.
- Error Logbook.
- Security Logbook.
- Document Logbook.
- Call Logbook.
- Equipment Logbook.
- CoDriver Logbook.

## Relationship to reports

Flow Reports explain executions.

Logbooks preserve the historical trail.

Flow Profiles learn from repeated reports.
